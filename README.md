# Patch-DELTARUNE-romana
Aplicație care modifică fișierele jocului DELTARUNE pentru a putea instala și juca modul mai ușor și mai rapid. Este făcut în limbajul de programare Python cu biblioteca `tkinter`


## Ce înseamnă fiecare variabilă
`foldel` - **fol**der **del**tarune<br>
`locfin` - **loc**ație **fin**ală<br>
`mesnuex` - **mes**ajul **nu** **ex**istă<br>
`jocfol` - **joc** **fol**der<br>
